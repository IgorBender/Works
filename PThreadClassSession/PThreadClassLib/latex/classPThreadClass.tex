\subsection{Detailed Description}
P\+O\+S\+IX thread class

Wrapper on P\+O\+S\+IX thread. The thread can be \+: ~\newline
 cyclic / on shot ~\newline
 attached / unattached ~\newline
 bound / unbound to L\+WP (for O\+Ss supporting this)
